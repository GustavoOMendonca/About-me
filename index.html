Projeto Principal

Agenda de Contatos

A Agenda de Contatos é um projeto desenvolvido em Java com o objetivo de praticar Programação Orientada a Objetos e manipulação de arquivos.

O sistema permite cadastrar, listar, buscar, editar e remover contatos, além de salvar as informações em um arquivo para que os dados não sejam perdidos ao fechar o programa.
